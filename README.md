# Guía de instalación del frontend

## Requisitos
- Tener Docker Desktop instalado.

## Instrucciones:

Desde la terminal ejecutar desde la raíz del proyecto: :

```docker-compose up --build```

## Acceso

Posterior a esto ya se podrá ingresar a la aplicación desde:

URL: 
`http://localhost:5173`

## Detener el contenedor: 

```docker-compose down```
